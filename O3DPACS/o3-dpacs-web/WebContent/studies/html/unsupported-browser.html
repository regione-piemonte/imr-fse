Something went wrong or this browser is not supported

It looks like you may be using a web browser version that in not supported
Viewer works best in the newest and last prior version of these browsers:

  • Google Chrome
  • Firefox
  • Apple Safari
  • Microsoft Edge

Viewer works also with Internet Explorer 11 but the performance can be lower.

Note: Any browser you use needs to have JavaScript turned on.

You can't update your browser

  • Ask your system administrator to update your browser if you cannot install updates yourself.
  • If you can't change your browser because of compatibility issues, think about installing a second browser for utilization of this site and keep the old one for compatibility.

Troubleshoot browser errors

If you're using a browser that works with Viewer but see the "This browser is not supported" error, an extension might be causing the problem. Try turning off your browser extensions.
